    Quick Start

  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get Polaris Clima up and running in minutes with our step-by-step guide.
  </p>

  <Steps>
    <Step title="Clone and install dependencies">
      Clone the repository and install all required packages:
      
      ```bash
      git clone https://github.com/Victor-Hernandez-Vivanco/Polaris-Clima.git
      cd Polaris-Clima
      pnpm install
      ```
    </Step>
    
    <Step title="Configure environment variables">
      Set up your database connection and environment configuration:
      
      ```bash
      # Create .env.local file
      DB_HOST=your_database_host
      DB_USER=your_database_user
      DB_PASSWORD=your_database_password
      DB_NAME=polaris_clima
      NEXT_PUBLIC_API_CONTACTO=/api/contacto
      ```
    </Step>
    
    <Step title="Run the development server">
      Start the Next.js development server:
      
      ```bash
      pnpm dev
      ```
      
      Open [http://localhost:3000](http://localhost:3000) to see your application.
    </Step>
    
    <Step title="Build and deploy">
      Build for production and deploy:
      
      ```bash
      pnpm build
      pnpm start
      ```
    </Step>
  </Steps>
</div>
