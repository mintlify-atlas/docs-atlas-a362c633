## Welcome to Polaris Clima

Polaris Clima is a modern, high-performance SaaS platform built for HVAC (Heating, Ventilation, and Air Conditioning) services in Santiago, Chile. The platform provides a complete digital presence for air conditioning installation, maintenance, and repair services with a focus on local SEO optimization and user experience.

## What is Polaris Clima?

Polaris Clima is a location-based service platform that helps customers find and request HVAC services in their specific comuna (district) in Santiago. The platform features:

- Dynamic location-based service pages for 17+ comunas in Santiago
- Professional service catalog (installation, preventive maintenance, corrective maintenance, technical visits)
- Integrated contact forms with MySQL database storage
- SEO-optimized pages with structured data (LocalBusiness and Service schemas)
- Responsive design with smooth animations powered by Framer Motion
- Static site generation for optimal performance

## Database Schema

The contact form stores submissions in a MySQL database:

```typescript title="lib/db.ts"
import mysql from 'mysql2/promise';

export async function getConnection() {
  return await mysql.createConnection({
    host: process.env.DB_HOST,
    user: process.env.DB_USER,
    password: process.env.DB_PASSWORD,
    database: process.env.DB_NAME,
  });
}
```

Table structure:
```sql
CREATE TABLE contactos (
  id INT AUTO_INCREMENT PRIMARY KEY,
  nombre VARCHAR(100) NOT NULL,
  apellido VARCHAR(100) NOT NULL,
  correo VARCHAR(150) NOT NULL,
  telefono VARCHAR(20) NOT NULL,
  mensaje TEXT,
  origen VARCHAR(50),
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```
