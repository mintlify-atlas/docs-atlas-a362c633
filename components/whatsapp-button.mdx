## Overview

The `WhatsAppButton` component is a fixed-position floating action button that provides direct WhatsApp contact functionality. Features include pulse animation, hover tooltip, customizable location-based messaging, and smooth hover effects.

<Warning>
  This is a **client component** (`'use client'`) for use in client-side contexts.
</Warning>

## Features

- Fixed bottom-right positioning
- Continuous pulse animation (CSS)
- Hover tooltip with message preview
- Optional location-based message customization
- WhatsApp deep link integration
- Smooth scale and shadow effects
- High z-index for persistent visibility

## Props

```tsx
interface Props {
  comuna?: string;  // Optional: Municipality/location name for message customization
}
```

<ParamField path="comuna" type="string" required={false}>
  Optional location identifier to customize the WhatsApp message.
  
  **Example:** `"Las Condes"`, `"Providencia"`, `"Vitacura"`
  
  When provided, the message becomes:
  ```
  Hola, estoy interesado en instalar aire acondicionado en {comuna}. ¿Podrían orientarme?
  ```
  
  When omitted, uses generic message:
  ```
  Hola, estoy interesado en instalar aire acondicionado. ¿Podrían orientarme?
  ```
</ParamField>

## Message Generation Logic

```tsx
const phoneNumber = '56979430510'; // Chile country code + number

const message = comuna
  ? `Hola, estoy interesado en instalar aire acondicionado en ${comuna}. ¿Podrían orientarme?`
  : 'Hola, estoy interesado en instalar aire acondicionado. ¿Podrían orientarme?';

const whatsappUrl = `https://wa.me/${phoneNumber}?text=${encodeURIComponent(message)}`;
```

<Accordion title="URL Structure Breakdown">
  - **Base URL:** `https://wa.me/`
  - **Phone Number:** `56979430510` (Chile +56, mobile 9 7943 0510)
  - **Query Parameter:** `?text=` + URL-encoded message
  - **Example:** `https://wa.me/56979430510?text=Hola%2C%20estoy%20interesado...`
</Accordion>

## Animation Setup

The pulse animation requires custom Tailwind configuration:

### tailwind.config.ts

```tsx
module.exports = {
  theme: {
    extend: {
      animation: {
        'scale-pulse': 'scalePulse 2s ease-in-out infinite',
      },
      keyframes: {
        scalePulse: {
          '0%, 100%': { transform: 'scale(1)' },
          '50%': { transform: 'scale(1.05)' },
        },
      },
    },
  },
};
```

### Alternative: Global CSS

If you prefer not to modify Tailwind config:

```css
/* globals.css */
@keyframes scalePulse {
  0%, 100% {
    transform: scale(1);
  }
  50% {
    transform: scale(1.05);
  }
}

.animate-scale-pulse {
  animation: scalePulse 2s ease-in-out infinite;
}
```

<Warning>
  Without this animation definition, the icon won't pulse. The component will still function but with reduced visual appeal.
</Warning>

## Z-Index Considerations

```tsx
z-50  // Very high z-index
```

**Common z-index hierarchy:**
- Modals: z-50 to z-60
- Fixed nav/headers: z-40
- Sticky elements: z-30
- WhatsApp button: z-50 (same as modals)

<Warning>
  If your app has modals or overlays with z-50 or higher, the WhatsApp button may appear above them. Consider using z-40 or adjusting modal z-index.
</Warning>

## Phone Number Configuration

The phone number is hardcoded in the component:

```tsx
const phoneNumber = '56979430510';
```

### To Change Phone Number:

<Steps>
  <Step title="Edit Component">
    Update the `phoneNumber` constant in `WhatsAppButton.tsx`
  </Step>
  
  <Step title="Format">
    Use international format without spaces or special characters:
    - Chile: `56979430510` (+56 9 7943 0510)
    - USA: `15551234567` (+1 555-123-4567)
    - Spain: `34612345678` (+34 612 34 56 78)
  </Step>
  
  <Step title="Test">
    Test the WhatsApp deep link in a browser to ensure it works
  </Step>
</Steps>

<Tip>
  For multi-tenant apps, consider making `phoneNumber` a prop or reading from environment variables.
</Tip>

## Example Implementation

Full example with all features:

```tsx
// app/servicios/[comuna]/page.tsx
import WhatsAppButton from '@/components/WhatsAppButton';

interface Props {
  params: { comuna: string };
}

export default function ServiciosComunaPage({ params }: Props) {
  const comunaName = params.comuna
    .split('-')
    .map(word => word.charAt(0).toUpperCase() + word.slice(1))
    .join(' ');
  
  return (
    <main>
      <h1>Servicios en {comunaName}</h1>
      
      <section>
        {/* Page content */}
      </section>
      
      {/* WhatsApp button with location */}
      <WhatsAppButton comuna={comunaName} />
    </main>
  );
}

// Example URLs:
// /servicios/las-condes → "Hola, estoy interesado... en Las Condes"
// /servicios/providencia → "Hola, estoy interesado... en Providencia"
```
