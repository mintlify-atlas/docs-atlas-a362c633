## Overview

The `TestimonialsSection` component displays customer testimonials with an intelligent responsive design: infinite horizontal carousel on mobile and a static grid on desktop. Features smooth auto-scroll, pause-on-interaction, and star ratings.

<Warning>
  This is a **client component** (`'use client'`) that uses React hooks for animation control.
</Warning>

## Features

- Infinite horizontal scroll on mobile (auto-animated)
- Static grid layout on desktop (1-3 columns)
- Pause animation on hover/touch
- Smooth RequestAnimationFrame-based animation
- Responsive column count based on testimonial quantity
- Star rating display
- Duplicate testimonials for seamless loop

## Component Architecture

### Mobile Layout (< 768px)

<Steps>
  <Step title="Duplicate Testimonials">
    Testimonials array is duplicated for seamless infinite loop:
    ```tsx
    const loopTestimonials = 
      testimonials.length > 1 
        ? [...testimonials, ...testimonials] 
        : testimonials;
    ```
  </Step>
  
  <Step title="Horizontal Scroll Container">
    Overflow container with hidden scrollbar:
    ```tsx
    <div
      ref={scrollRef}
      className="flex gap-4 overflow-x-scroll no-scrollbar"
    >
    ```
  </Step>
  
  <Step title="Auto-Scroll Animation">
    RequestAnimationFrame-based smooth scrolling at 0.3px/frame
  </Step>
  
  <Step title="Loop Reset">
    When scrollLeft reaches halfway (totalWidth/2), reset to start:
    ```tsx
    if (container.scrollLeft >= totalWidth) {
      container.scrollLeft -= totalWidth;
    }
    ```
  </Step>
</Steps>

### Desktop Layout (≥ 768px)

Static grid with responsive column count:

```tsx
const cols =
  testimonials.length === 1
    ? 'md:grid-cols-1'
    : testimonials.length === 2
      ? 'md:grid-cols-2'
      : 'md:grid-cols-3';

<div className={`hidden md:grid grid-cols-1 ${cols} gap-6`}>
  {testimonials.map((testimonial, index) => (
    <TestimonialCard key={index} {...testimonial} />
  ))}
</div>
```

## Animation System

### useEffect Hook

```tsx
useEffect(() => {
  const container = scrollRef.current;
  if (!container || testimonials.length <= 1) return;

  let animationFrameId: number;
  const speed = 0.3; // Pixels per frame

  const startAnimation = () => {
    const totalWidth = container.scrollWidth / 2;

    const animate = () => {
      if (!isPaused) {
        container.scrollLeft += speed;

        if (container.scrollLeft >= totalWidth) {
          container.scrollLeft -= totalWidth;
        }
      }

      animationFrameId = requestAnimationFrame(animate);
    };

    animationFrameId = requestAnimationFrame(animate);
  };

  const timeout = setTimeout(startAnimation, 300); // Delay start

  return () => {
    cancelAnimationFrame(animationFrameId);
    clearTimeout(timeout);
  };
}, [isPaused]);
```

### Pause on Interaction

```tsx
const [isPaused, setIsPaused] = useState(false);

<div
  onMouseEnter={() => setIsPaused(true)}
  onMouseLeave={() => setIsPaused(false)}
  onTouchStart={() => setIsPaused(true)}
  onTouchEnd={() => setIsPaused(false)}
>
```

<Accordion title="Why RequestAnimationFrame?">
  - Smooth 60fps animation
  - Automatically pauses when tab is inactive
  - Better performance than setInterval
  - Synchronized with browser repaint cycle
</Accordion>

## Responsive Behavior

| Breakpoint | Layout | Interaction | Column Count |
|------------|--------|-------------|-------------|
| < 768px    | Horizontal carousel | Auto-scroll + swipe | N/A |
| ≥ 768px    | Grid | Static | 1-3 (dynamic) |

### Dynamic Column Count Logic

## Performance Optimization

<AccordionGroup>
  <Accordion title="RequestAnimationFrame">
    Uses browser-optimized animation API instead of setInterval:
    - Automatically pauses when tab is inactive
    - Syncs with display refresh rate (60fps)
    - Lower CPU usage
  </Accordion>
  
  <Accordion title="Conditional Duplication">
    Only duplicates testimonials array when > 1 item:
    ```tsx
    const loopTestimonials =
      testimonials.length > 1 
        ? [...testimonials, ...testimonials] 
        : testimonials;
    ```
  </Accordion>
  
  <Accordion title="Early Return">
    Animation doesn't run if only 1 testimonial:
    ```tsx
    if (!container || testimonials.length <= 1) return;
    ```
  </Accordion>
  
  <Accordion title="Cleanup">
    Properly cancels animation on unmount:
    ```tsx
    return () => {
      cancelAnimationFrame(animationFrameId);
      clearTimeout(timeout);
    };
    ```
  </Accordion>
</AccordionGroup>

## Customization

### Adjust Scroll Speed

```tsx
const speed = 0.3; // Default: 0.3px per frame

// Slower (smoother):
const speed = 0.15;

// Faster:
const speed = 0.5;
```

### Change Animation Delay

```tsx
const timeout = setTimeout(startAnimation, 300); // Default: 300ms

// Start immediately:
const timeout = setTimeout(startAnimation, 0);
```

### Modify Card Width (Mobile)

```tsx
<div className="min-w-[85%]"> {/* Default: 85% */}

// Wider cards:
<div className="min-w-[95%]">

// Narrower cards (shows more cards):
<div className="min-w-[75%]">
```

### Change Background Color

```tsx
<section className="bg-blue-50"> {/* Default */}

// White background:
<section className="bg-white">

// Gray background:
<section className="bg-gray-100">
```
