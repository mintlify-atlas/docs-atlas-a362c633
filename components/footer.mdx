## Overview

The `Footer` component provides a complete footer section with navigation links, service listings, contact information, social media integration, and a responsive mobile/desktop layout. Includes smart scroll-to-top behavior for the logo.

<Warning>
  This is a **client component** (`'use client'`) that uses Next.js routing hooks.
</Warning>

## Features

- Four-column grid layout (desktop)
- Responsive mobile layout with bottom logo bar
- Smart logo click behavior (scroll vs navigate)
- Social media links (Instagram, WhatsApp)
- Contact information with icon integration
- Section anchor links
- Hover animations on all interactive elements

## Usage

```tsx
import Footer from '@/components/Footer';

export default function RootLayout({ children }) {
  return (
    <html>
      <body>
        {children}
        <Footer />
      </body>
```

## Smart Logo Click Handler

```tsx
import { useRouter, usePathname } from 'next/navigation';

const router = useRouter();
const pathname = usePathname();

const handleLogoClick = (e: React.MouseEvent) => {
  e.preventDefault();

  if (pathname === '/') {
    // Already on homepage: scroll to top
    window.scrollTo({ top: 0, behavior: 'smooth' });
  } else {
    // Different page: navigate home
    router.push('/');
  }
};
```

<Card title="Behavior" icon="arrow-pointer">
  - **On homepage:** Smoothly scrolls to page top
  - **On other pages:** Navigates to homepage
  - Prevents default link behavior
</Card>

## Link Hover Animations

All footer links use consistent hover effects:

```tsx
className="
  inline-block              // Required for transform
  text-black               // Base color
  text-xs md:text-sm       // Responsive sizing
  transition-all           // Smooth transitions
  duration-300             // 300ms duration
  ease-out                 // Ease-out timing
  hover:scale-110          // Scale up on hover
  origin-left              // Scale from left edge
"
```

<Tip>
  The `origin-left` class ensures links scale from their left edge, creating a more natural animation than the default center origin.
</Tip>

```tsx
<div className="max-w-[1400px] mx-auto">
```

### Pointer Events

Interactive elements have `pointer-events-auto`:

```tsx
<nav className="pointer-events-auto ...">
```

This ensures links are clickable even if parent has `pointer-events-none`.
