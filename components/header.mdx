## Overview

The `Header` component provides the main navigation for the Polaris Clima website. It features a sticky header with glassmorphism effects, scroll-based transformations, and a responsive mobile menu.

**Source**: `~/workspace/source/components/Header.tsx`

## Features

- **Sticky Navigation**: Fixed positioning that follows the user as they scroll
- **Scroll Effects**: Transforms from full-width to rounded pill shape when scrolled
- **Glassmorphism**: Backdrop blur with semi-transparent background
- **Mobile Menu**: Hamburger menu for small screens with slide-in drawer
- **Smart Logo Behavior**: Scrolls to top on home page, navigates on other pages
- **Anchor Link Handling**: Smooth scroll to sections on home page

### Scroll Detection

The component uses a scroll event listener to detect when the user has scrolled past 80px:

```tsx
const [isScrolled, setIsScrolled] = useState(false);

useEffect(() => {
  const onScroll = () => {
    setIsScrolled(window.scrollY > 80);
  };

  window.addEventListener('scroll', onScroll);
  return () => window.removeEventListener('scroll', onScroll);
}, []);
```

### Smart Logo Click Handler

The logo behavior changes based on the current route:

```tsx
const handleLogoClick = (e: React.MouseEvent) => {
  e.preventDefault();

  if (pathname === '/') {
    // On home page: scroll to top
    window.scrollTo({ top: 0, behavior: 'smooth' });
    
    // Clear any hash in URL
    if (window.location.hash) {
      window.history.replaceState(null, '', '/');
    }
  } else {
    // On other pages: navigate to home
    window.location.assign('/');
  }
};
```

### Anchor Link Navigation

For section links on the home page, the component implements smooth scrolling:

```tsx
const handleNavClick = (e: React.MouseEvent, href: string) => {
  e.preventDefault();

  // If not on home page, perform full navigation
  if (pathname !== '/') {
    window.location.assign(href);
    return;
  }

  // If on home page, smooth scroll to section
  const id = href.replace('/#', '');
  const section = document.getElementById(id);

  if (section) {
    section.scrollIntoView({ behavior: 'smooth' });
    window.history.replaceState(null, '', href);
  }
};
```

## Responsive Design

### Desktop Navigation (≥768px)

- Full horizontal navigation menu
- Logo on left, nav links in center, contact button on right
- Larger logo size (48px height)
- Wider padding and margins

### Mobile Navigation (&lt;768px)

- Hamburger menu icon (☰)
- Smaller logo (32px height)
- Full-screen slide-in drawer menu
- Stacked vertical navigation links

### Mobile Menu Structure

```tsx
<div className={`
  fixed inset-0 z-40 bg-white
  transform transition-transform duration-300
  ${isMenuOpen ? 'translate-x-0' : 'translate-x-full'}

    <Link href="/#soluciones" onClick={handleNavClick}>Soluciones</Link>

  </div>

```

## Performance Considerations

<Warning>
The component uses `window.location.assign()` for non-home page navigation instead of Next.js `<Link>` for static export compatibility. This causes full page reloads when navigating between pages.
</Warning>

**Optimization notes:**
- Scroll listener is properly cleaned up to prevent memory leaks
- Logo is a standard `<img>` tag (not Next.js Image) for immediate rendering
- Transitions are GPU-accelerated via transform properties
